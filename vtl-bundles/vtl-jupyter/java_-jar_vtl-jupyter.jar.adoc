== Options

*-e*, *--exec* _<exec>_::
  Run the VTL E&E kernel. Should only be called by Jupyter.

*-h*, *--help*::
  Show this help message and exit.

*-i*, *--install*::
  Installs this kernel into the Jupyter data path.

*--jupyter* _jupyter_::
  Location of Jupyter executable.

*-V*, *--version*::
  Print version information and exit.
